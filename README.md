## Installation
To set up this project on your local machine, follow these steps:

```
git clone https://github.com/rohan-singh987/zag-task1

# Navigate to the project directory:

# Install dependencies:
npm install
yarn add

```

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
